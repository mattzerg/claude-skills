# Swag Placement Gate

Run this before presenting swag mockups or vendor files.

## High-Risk Failures

- Logo is pasted into an arbitrary empty corner instead of an item-specific brand zone.
- Full logo and campaign graphic compete at the same hierarchy level.
- Logo appears on top of seams, zipper, drawstrings, pocket opening, handle, brim curve, or folded fabric without intentional treatment.
- Text is technically correct in source but visually illegible, clipped, merged, too small, or distorted in the preview.
- Print method cannot reproduce the detail: embroidery text too small, woven patch too detailed, mug wrap too close to handle, sticker cutline too fragile.
- Same placement logic is copied across different item types without adapting to anatomy.
- Review PDF shows only flat artwork and not the artwork on actual item silhouettes.

## Required Checks

1. Full-logo check:
   - If required on every item, identify the exact full-logo zone per item.
   - Confirm the full logo reads at the displayed size.
2. Hero/secondary hierarchy:
   - State which element is hero and which is secondary.
   - If both feel equal, reduce one.
3. Physical plausibility:
   - Check item anatomy, object curvature, fabric folds, seams, closures, handles, and print zones.
4. Legibility:
   - Inspect at review PDF size and likely real-world size.
   - Confirm exact text is readable and not clipped.
5. Vendor feasibility:
   - Note print method and any detail simplification needed.
6. Package completeness:
   - Include item mockups first, production artwork second, and a vendor note/spec.

If any high-risk failure is present, revise before presenting unless the user explicitly asks to see rough work.
