## Out-of-scope (deliberately excluded)

- **Cold outbound to journalists** — different motion, owned by `zerg-prospecting` skill or human relationships
- **Paid acquisition** (Google Ads, LinkedIn Ads, Twitter Ads) — separate playbook, not part of launch campaign
- **Affiliate programs** — Phase 3+
- **Press releases via wire services** — low signal, expensive

## How to use this registry

1. `launch-campaign plan <slug>` reads this file, picks tiers appropriate for the launch (e.g., dev-tool launches hit all 6; consumer-facing skip Tier 6), and emits a per-launch plan
2. Plan output references this file's URLs + lead times verbatim
3. Verify-flagged channels show up in the plan with `[VERIFY URL ACTIVE BEFORE SUBMIT]` next to them
4. When a channel goes dark / changes URL / new entry deserves inclusion, edit this file directly
