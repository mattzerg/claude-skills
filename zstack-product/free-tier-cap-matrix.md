# Free-Tier Cap Matrix — Zstack Microproducts

The vault doesn't centralize this. The skill ships it, and the audit enforces it.

Free tier is volume-capped per product, not feature-capped. Upgrade unlocks volume + agent governance, NOT new core features.

| Product | Free cap | Pro unlocks |
|---------|----------|-------------|
| Zergboard | 3 active users | unlimited users, scoped agent tokens, SSO |
| ZergChat | 3 active users | unlimited users, agent governance |
| ZergCal | 1 calendar / 1 event type per user | unlimited calendars + event types |
| ZergMeeting | 5 meetings/month | unlimited meetings |
| ZergMail | 50 messages/day | unlimited (subject to deliverability throttle) |
| ZergCRM | 100 contacts | unlimited contacts |
| ZergSend | 1,000 contacts | unlimited contacts |
| ZergAlytics | 10K events/month | unlimited events, per-agent attribution |
| Zergwallet | 1 connected account | unlimited accounts, agent autonomy tier |

## Bundle discount math (Pro tier)

- 5×$9 = $45 individually
- Zstack Pro = $19 → 58% discount
- Pitch: "Replace Linear, Slack, Calendly, Otter, and Front for $19/seat/mo"

## Audit rule

- Positioning doc must declare cap matching this matrix verbatim. Drift = MED.
- New products must add a row before bootstrap completes.
