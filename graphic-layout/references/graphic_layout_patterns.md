# Graphic Layout Patterns

These are the local composition templates the skill can rely on even when a vault-level `graphic_layout.md` file is absent.

## Global rules

- One dominant focal element per asset
- Top and bottom whitespace should feel intentional and roughly balanced
- Empty space must frame the composition, not look like unused canvas
- Off-center layouts should be visibly intentional, not "title on one side, filler on the other"
- Repeated assets in a campaign should share anchor points and structural rhythm

## Template: Hero

- Canvas: 1200x630
- Structure: headline centered or intentional rule-of-thirds anchor with one supporting visual
- Anti-pattern: weak eyebrow stealing top-band height without adding information

## Template: Split Comparison

- Canvas: 1200x630 or 1200x675
- Structure: equal visual weight left and right
- Anti-pattern: one side is real content, the other is decorative empty space

## Template: Stat Strip

- Structure: centered title, 3 equal stat blocks
- Anti-pattern: cards with unequal width or irregular vertical alignment

## Template: Annotated Screenshot

- Structure: product pane + callout rail
- Anti-pattern: labels floating directly over product UI without visual separation

## Template: Title Card / End Card

- Structure: brand, one headline, one CTA, one support element
- Anti-pattern: too many competing text regions or decorative clutter
