# Shot Grammar

> **For measured numbers** (per-beat duration budgets, cut cadence/MSL by format, named ordering patterns, b-roll-to-hero ratio), use `sequence-techniques.md` — it supersedes the prose below wherever a measured value exists. This file names the grammar; the techniques catalog times it.

## Product UI

- Establish: product surface or problem state.
- Input: user action, command, import, question, or trigger.
- Processing: short progress moment only if it builds trust.
- Output: visible result, before/after, or saved time.
- Detail: zoom/callout for the important UI state.
- Human context: where the result goes next.

For software launch videos, the UI sequence must be cinematic proof, not a tutorial. Sequence the causal chain explicitly:

- Before state.
- Trigger/input.
- Product or agent action.
- After state.
- Verification surface: feed, count, log, notification, audit row, or human handoff.

If one of those steps is missing, the viewer will feel the product claim is fake or vague.

Shot composition modes:

- Hero surface: full product reveal or outcome. Use sparingly.
- Macro proof: crop to the active workflow region.
- Split causal chain: input and output visible together.
- Exploded UI layer: pull one object out and attach metadata around it.
- Montage tile: 3-4 quick capability tiles, one verb each.
- End card: brand, promise, CTA, no competing UI.

Avoid repeating the same full-board composition across multiple beats.

## Launch / Reveal

- Pattern interrupt or pain image.
- Title or product name.
- Hero capability.
- Two proof beats.
- Credibility beat.
- CTA card.

Short launch clips need a turn before the midpoint. The turn can be a problem collapsing into the product, a card moving to the correct state, a messy workflow becoming legible, or a human seeing what an agent did.

## Founder / Talking Head

- Cold open close-up.
- Context b-roll.
- Product proof insert.
- Return to speaker for conviction or story turn.
- CTA or invitation.

## Case Study

- Customer environment.
- Before workflow.
- Constraint or failure point.
- Intervention.
- Result proof.
- Customer quote or team reflection.

## Brand Film

- World or audience context.
- Tension.
- Belief statement.
- Evidence montage.
- Invitation.

## Social / Paid

- First-frame curiosity.
- Visible transformation.
- Short proof.
- CTA.
